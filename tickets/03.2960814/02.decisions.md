# 02. Decisions

## Decision Log
- Owner: marinRoglev (Solution Architect, AWS certified)
- Migration involves 470M+ records, requiring robust planning
- Use Zod for code-based schema management (chosen for type safety and testability)
- Maintain .dbml for ERD and visualization (chosen for clarity and cross-team communication)
- Separate schema evolution from migration execution (reduces risk and avoids blocking delivery)

## Alternatives Considered
- Manual schema management (rejected: higher risk, less traceability)
- Direct migration scripts before schema validation (rejected: risk of errors and rework)
